---

### 🎨 THE MANIFESTO (My Development Rules)

1. **SOLID over Sloppy**: Structure is everything. Decouple your business logic, implement SRP, and design for extension.
2. **Strict Type Safety**: Reject `any`. Compile-time safety is the ultimate guardrail against runtime panics.
3. **No Boring UIs**: Give me heavy borders, high-contrast solid drop shadows, retro palettes (Gruvbox), and responsive micro-animations.
4. **Performance is Paramount**: Build interactive experiences, but optimize where it counts (e.g. JIT DOM splitting to prevent layout freezes).

### 🚀 Highlight Projects

#### 🎮 [Break The Web](https://github.com/andreyyste/break-the-web)
*A meta-game where a standard React portfolio website becomes the physics layout for a 2D Phaser game.*
* **Highlights:** Integrated a React-Phaser event bridge allowing elements like inputs (typing passcodes to open vaults) and profile pictures (spawning as 100kg boulders to press plates) to cross realms into the game canvas.
* **Tech:** React 19, Phaser 3, Zustand, Vanilla CSS.

#### 🐍 [Snakey & Snakey Extension](https://github.com/andreyyste/snakey)
*A Chrome extension that releases a snake onto any active webpage to eat HTML elements.*
* **Highlights:** Implemented Proximity-Based JIT (Just-In-Time) Splitting. Keeps rendering lag-free on heavy pages by only splitting text paragraphs when the snake head is within a 100px buffer (letter-by-letter for headers, word-by-word for paragraphs).
* **Tech:** React 19, Tailwind CSS v4, Phaser 3/4, Manifest V3.

#### 🤠 [RDR2 Pause Menu Academic Hub](https://github.com/andreyyste/rdr2-menu)
*An academic dashboard and GPA progress tracker styled around the Red Dead Redemption 2 pause menu.*
* **Highlights:** Real-time analog pocket watch, SKS/GPA weighted calculators, and a custom calendar board with multiple themes (*Classic*, *Blood*, *Sand*).
* **Tech:** HTML5, CSS3, Vanilla JavaScript, LocalStorage API.

#### 🎨 [Portfolio Neo-Gruv](https://github.com/andreyyste/portofolio-neo-gruv)
*Personal portfolio built using Next.js 15, rejecting standard minimalist designs for a bold Neo-Brutalist look.*
* **Highlights:** Embedded codebase explorer utilizing Shiki syntax highlighting, Next.js edge JWT cookie parsing, and automatic 401 session eviction.
* **Tech:** Next.js 15, NestJS, Prisma ORM, PostgreSQL.

#### 🎓 [StudentHub UGM](https://github.com/andreyyste/student-hub)
*A productivity tool for UGM students integrating lecture scheduling, task lists, and offline document storage.*
* **Highlights:** Intercepts eLOK portal document downloads, stores data in SQLite, and features an interactive AI chatbot supporting Google Gemini, Local Ollama, and GitHub Models.
* **Tech:** Flutter (Dart), SQLite, InAppWebView.
